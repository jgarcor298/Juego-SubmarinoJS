# Juego-SubmarinoJS
**Autor:** Jorge Garre Corrales

**Descripción:** Juego del Submarino en JS, se despliega un tablero y el usuario clicka un boton con el que aparece una ventana modal que pide al usuario las coordenadas donde quiere disparar, en caso de acierto se lo hace saber al usuario por pantalla y en caso de fallo, también avisa al usuario y el submarino.


**Problemas encontrados al desarrollar la práctica:** No he sido capaz de ponerle un addEventListener a cada celda para poder saber que celda clicka el usuario y comprobar la ubicación del submarino
